# jekyll-docker-site

Base template for a Jekyll site using Docker

## Installation

- Install these things:
  - Visual Studio Code
  - Docker
  - Visual Studio Code extensions
    - Docker
    - Dev Containers

- clone this template
- git clone this repo to your computer
